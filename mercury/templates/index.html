NYU Motorsports Telemetry

Telemetry platform for monitoring NYU Tandon's Motorsports BAJA SAE competition vehicle. This platform supports visualization of data from vehicle sensors along with several auxiliary features.

Dashboard

Opens a new window containing the telemetry visualization tool, which queries the database to populate several graphs. Graphs are fully configurable and can be added or deleted as needed.

Stopwatch

Track overall race time and individual laps in the browser.

Raw Telemetry

View raw data row-by-row as it appears in the database tables.

Simulator

Generates mock sensor data and sends it to the database tables.

CAN Decoder

Input CAN messages and view the return values.
